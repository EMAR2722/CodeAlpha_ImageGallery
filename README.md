# My Memories – Premium Image Gallery

A modern, interactive gallery to showcase life’s best moments. Built with vanilla HTML, CSS, and JavaScript.

## ✨ Features

- **Category filtering** – All, Childhood, My Wedding, Graduation, My Family, Nature  
- **Premium lightbox** – Full‑screen view with counter, caption, and navigation that respects filters  
- **Keyboard support** – Arrow keys to navigate, ESC to close  
- **Responsive grid** – Adapts from 1 to 3+ columns  
- **Smooth hover effects** – Zoom & overlay with image title  
- **Dark glassmorphism theme** – Modern, elegant look  

---

## 🚀 Getting Started

1. Clone or download the repository.  
2. Open `index.html` in any modern browser.  
3. Click filter buttons to explore categories, click images to open the lightbox.

No build steps or dependencies required.
